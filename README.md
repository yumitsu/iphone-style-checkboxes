# iOS Checkboxes

Turn your checkboxes into iPhone-style binary switches.

Homepage:: http://ios-checkboxes.awardwinningfjords.com/

## Use

### Pure JS:

```javascript
iOSStyle(document.getElementById('checkbox'))
```

### jQuery:

```javascript
$('#checkbox').iOSStyle();
```
